# Samsung Download Mode Script

A very simple Python script to boot Samsung devices into **Download Mode** on ZC builds and future builds.

## Requirements

- Python 3
- pySerial

## Install pySerial

Install the required package with pip:

```
pip install pyserial
```

## Usage

1. Connect your phone (make sure it is unlocked from the lockscreen)
2. Run the script
3. Choose option you want to use

## Notes

- If on Windows, make sure to have the Samsung USB Drivers installed.
